# LED AppBar

## Versioning

Update all three files when bumping the version:

- `package.json` — `version`
- `src-tauri/Cargo.toml` — `version`
- `src-tauri/tauri.conf.json` — `version`

## README sync

`README.md` (English) and `README.ja.md` (Japanese) must always be kept in
content-sync. When updating one, apply the equivalent change to the other in
the same commit. Do not let them drift.
